Comparison between DOMContentLoaded and load

不懂php也没关系，sleeptime=10表示至少等待10s才能被加载成功

DOMContentLoaded say: 图片的加载与我无关
load say: 图片是我兄弟，无论如何也要等图片加载完成

html

DOMContentLoaded say: 我只等待样式表的加载，但不会考虑样式表所引用的文件
load say: 呵呵，我和你有点类似，我也等待样式表的加载，但也要等待样式表里所引用的部分文件的加载。比如DOM元素需要用到的背景图。
